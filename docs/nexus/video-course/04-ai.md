# 04 — LLM-led navigation, safely

## Shot list

1. **00:00 — Cold open.** An LLM picks "click the Save button" from a
   cluttered admin screen.
2. **01:00 — Architecture.** Navigator → VisualContext → LLM → JSON
   action → Engine.Do.
3. **02:30 — Budget first.** Live-code `NewCostTracker(1.0)`. Show
   `Reserve(2.0)` returning `ErrBudgetExceeded`.
4. **04:00 — Navigator.** Call `Decide(ctx, vc)`; show the JSON
   contract and the tolerant parser.
5. **06:00 — Healer.** Inject a selector drift; show the healer
   returning `aria-label="Save"` as the replacement.
6. **08:30 — Generator.** Paste a user story in a text file; watch the
   generator emit valid bank YAML; run it.
7. **11:00 — Predictor.** Train on a tiny history; show how the
   threshold knob tunes false-positives.
8. **13:00 — Outro + safety reminder.** Temperature 0 for deterministic
   flows; budget enforced; no inline-script execution allowed.
