# Helix Nexus — Accessibility

`pkg/nexus/a11y` injects a vendored axe-core bundle into the running
browser Engine and parses the JSON verdict into typed `Violation` rows.
`Report.Assert(LevelA|AA|AAA)` returns a non-nil error when any
violation at or above the required severity is present. Section 508 is
filtered from `Report.Section508()` so compliance reports can list the
applicable subset independently.

## Vendoring axe-core

Ship `axe.min.js` under `docs/nexus/vendor/`. Tests use a local
httptest server that serves the file; production installs place it
behind the operator's static asset CDN. `InjectionScript(assetBase)`
returns the JS Nexus should evaluate through the browser Engine.

## Severity mapping

| Level | Breach on |
|---|---|
| A   | any **critical** |
| AA  | any **critical** or **serious** |
| AAA | any **critical**, **serious**, or **moderate** |

Minor violations are tracked but never fail the build at any level.
