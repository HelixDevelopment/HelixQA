# HelixQA v4.0.0 ‚Äî OpenClaw Ultimate

**Release date:** 2026-04-18
**Headline:** The full OCU program (P0 ‚Üí P7) is closed. HelixQA gains a
native capture / vision / interact / observe / record stack behind a
unified automation engine, with LLM-sole-decider invariants intact and
10-category test coverage green across all OCU packages.

---

## What landed (P0 ‚Üí P7)

### P0 ‚Äî Foundation + Go‚ÜîNative bridging

- `pkg/nexus/native/contracts/` ‚Äî six locked contract interfaces:
  `CaptureSource`, `VisionPipeline`, `Interactor`, `Observer`,
  `Recorder`, `Remote`. Version v1 (frozen for the lifetime of the
  v4.x line).
- `pkg/nexus/native/budget/` ‚Äî `Budget` struct + per-call accounting.
- `pkg/nexus/native/probe/` ‚Äî `ProbeLocal()` GPU/CPU/RAM capability
  probe; latency ~133 ¬µs on i7-1165G7.
- `pkg/nexus/native/remote/` ‚Äî `Dispatcher` for thinker.local RPC.
- Containers GPU extension: `HostResources.GPU`, `GPURequirement`,
  `StrategyGPUAffinity`, `GPUHealthCheck`, `ProbeGPU`.
- `pkg/nexus/native/contracts/contracts_test.go` ‚Äî compile-time
  interface-satisfaction checks (A1‚ÄìA6).
- Bank: `banks/ocu-foundation.json` (10 entries).

### P1 ‚Äî GPU capture engine

- Factory + 4 `CaptureSource` backends:
  - `web/` ‚Äî CDP-planned (injectable `newFrameProducer`).
  - `linux/` ‚Äî X11 SHM-planned.
  - `android/` ‚Äî ADB screenrecord (phone + TV).
  - `androidtv/` ‚Äî reuses android source; TV-specific config.
- Stress tested: 100-goroutine -race clean.
- Per-source bench: ~10 ms Open‚Üíframe‚ÜíClose on stub producer.
- Bank: `banks/ocu-capture.json`.
- Production subprocess wiring deferred to P1.5 via injectable sentinel.

### P2 ‚Äî GPU vision pipeline

- `VisionPipeline` with four operations: `Analyze`, `Match`, `Diff`, `OCR`.
- `pkg/nexus/vision/cpu/` ‚Äî CPU local backend (pure-Go stubs).
- Remote-dispatch plumbing via `ocuremote.Dispatcher`.
- Bench: Analyze ~3.8 ns/op 0 allocs; Diff ~48 ns/op 1 alloc (stub).
- Bank: `banks/ocu-vision.json` (13 entries).
- Real OpenCV CUDA + TensorRT OCR deferred to P2.5.

### P3 ‚Äî Interaction engine (kernel-level)

- Factory + 4 `Interactor` backends:
  - `linux/` ‚Äî uinput-planned (injectable `newInjector`).
  - `web/` ‚Äî CDP-planned.
  - `android/` ‚Äî ADB input (phone).
  - `androidtv/` ‚Äî ADB input (TV / DPAD).
- `pkg/nexus/automation/verifier/` ‚Äî `Verifier` hook interface + `Wrap`
  adapter + `NoOp`. Bench: Wrap_Click ~86 ns/op 0 allocs.
- Bank: `banks/ocu-interact.json` (19 entries).
- Real uinput/CDP/ADB wiring deferred to P3.5.

### P4 ‚Äî Observation engine (hook-based)

- Factory + 5 `Observer` backends:
  - `ld_preload/` ‚Äî dynamic linker preload hook.
  - `plthook/` ‚Äî PLT entry patching.
  - `dbus/` ‚Äî D-Bus signal listener.
  - `cdp/` ‚Äî Chrome DevTools Protocol event stream.
  - `ax_tree/` ‚Äî Accessibility tree walker.
- Shared `BaseObserver` + `RingBuffer` (thread-safe, configurable capacity).
- Injectable `producerFunc` sentinel; production returns `ErrNotWired`.
- Bank: `banks/ocu-observe.json` (21 entries).
- Real shim/hook/bus wiring deferred to P4.5.

### P5 ‚Äî Recording and streaming

- `Recorder` with three encoder stubs: x264, nvenc, vaapi.
- `FrameRing` ‚Äî lock-free ring buffer for in-flight frames.
- Clipper: ND-JSON output (MKV header deferred to P5.5).
- Priority-drain goroutine prevents frame loss on `Stop()`.
- WebRTC/WHIP publisher off by default (`ErrNotWired`).
- 100-goroutine stress -race clean.
- Bank: `banks/ocu-record.json` (21 entries).
- FFmpeg/NVENC CGO + real WHIP deferred to P5.5.

### P6 ‚Äî Unified automation surface

- `pkg/nexus/automation/Engine` ‚Äî composes P1‚ÄìP5 behind one
  `Perform(ctx, Action) (Result, error)` call.
- Eight `ActionKind` constants (click, type, scroll, key, drag, capture,
  analyze, record_clip).
- `Result` with `Success`, `VerificationPassed`, `Elapsed`, `Evidence`,
  `Error`, `DispatchedTo`.
- `EvidenceRef{Kind, Ref}` ‚Äî stable key for evidence-store resolution.
- `pkg/nexus/automation/agent_bridge/Bridge` ‚Äî one-line adapter into
  `pkg/nexus/agent`. LLM remains sole decider; Engine is pure dispatcher.
- `PixelVerifier` + `MultiVerifier` (post-action screen validation).
- 100-goroutine stress -race clean.
- Bank: `banks/ocu-automation.json` (22 entries).

### P7 ‚Äî Magical tickets + tests + challenges (this release)

- `pkg/ticket/ocu_evidence.go` ‚Äî 12 `EvidenceKind*` string constants:
  `clip`, `diff_overlay`, `ocr_dump`, `element_tree`, `hook_trace`,
  `replay_script`, `llm_reasoning`, `perf_metrics`, `ax_tree_diff`,
  `har`, `webrtc_stream`, `raw_dma`. Plus `Evidence{Kind, Ref}` struct.
- `pkg/ticket/ocu_mapping.go` ‚Äî `FromAutomationResult(Result) []Evidence`
  translates automation EvidenceRefs into ticket Evidence entries.
- `pkg/ticket/ocu_replay.go` ‚Äî `BuildReplayScript([]Action) string`
  emits `.ocu-replay` DSL for `helixqa replay <ticket>`.
- `docs/ocu-replay-format.md` ‚Äî DSL specification v1.0.
- `scripts/ocu-full-campaign.sh` ‚Äî 8-category campaign runner (unit,
  integration, stress, govulncheck, vet, bench, gofmt, challenges).
- Four cross-cutting challenge banks (81 new entries):
  - `banks/ocu-tickets.json` ‚Äî 36 entries (12 kinds √ó 3 cases).
  - `banks/ocu-adversarial.json` ‚Äî 20 adversarial entries.
  - `banks/ocu-cross-platform.json` ‚Äî 15 cross-platform entries.
  - `banks/ocu-fixes-validation.json` ‚Äî 10 regression guards.
