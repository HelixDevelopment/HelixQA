# HelixQA website

VitePress source for the public HelixQA documentation site.

```bash
# install
npm install

# dev
npm run docs:dev

# build
npm run docs:build
```

The `/nexus` section is the source-of-truth for Helix Nexus marketing
and long-form content. Every sub-page corresponds to a shipped Nexus
package (`pkg/nexus/*`).
